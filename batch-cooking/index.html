Assistant cuisine

Choisissez votre point de depart, puis construisez la semaine autour.

Ce prototype est pense autour de deux vrais points d'entree : "je sais deja quel plat je veux manger cette semaine" ou "je dois utiliser ces ingredients avant qu'ils ne s'abiment". A partir de la, le planificateur construit les composants, les repas, les courses, les substitutions et le stockage.

Construire la semaine Retour aux projets

Ce que cherche a faire le planificateur

  • Ancrer la semaine : autour d'un plat souhaite ou de trois ingredients deja disponibles.
  • Reutiliser intelligemment : un petit ensemble de proteines, feculents, legumes et sauces.
  • Reduire la friction : une liste de courses, une session du dimanche, plusieurs resultats.
  • Garder le controle : substitutions, fraicheur et stockage restent visibles.

04 / Moteur de substitutions

Des substitutions qui conservent une vraie logique nutritionnelle

Les substitutions mettent a jour les quantites et les macros, pas seulement les noms des ingredients.

Note nutritionnelle

Ce prototype utilise des equivalences simples pour montrer le comportement attendu.

  • Substitutions de proteines : viser un apport proteique proche par portion.
  • Substitutions de glucides : garder la structure du repas tout en ajustant les calories.
  • Substitutions de richesse : conserver la sensation du plat, pas seulement le nom de l ingredient.
